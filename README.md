# 🛰 Project Sentinel

> Cybersecurity Engineering Workspace

Project Sentinel is my personal Cybersecurity Engineering Workspace, designed to document real projects, laboratory environments, technical decisions and continuous learning throughout my professional journey.

It was created from the need to build a structured, practical and continuously evolving learning environment capable of supporting my long-term professional growth.

More than a study repository, this workspace documents real projects, engineering decisions, laboratory environments and technical evolution across Cybersecurity, Infrastructure, Automation and Software Development.

## 🌍 Vision

To evolve from an Infrastructure and Cybersecurity professional into a Cybersecurity Engineer and, ultimately, a Cybersecurity Architect through practical experience, engineering mindset and continuous learning.

## 📖 About

Instead of being a collection of isolated study notes, Project Sentinel serves as an engineering workspace where every lab, project and technical decision is documented.

Each Sprint represents a new milestone in my professional growth.

The long-term goal is to evolve professionally towards Cybersecurity Engineering and later Cybersecurity Architecture.

 ## 🎯 Objectives 

- Develop practical cybersecurity skills
- Create engineering projects based on real-world scenarios
- Learn through hands-on labs
- Document technical decisions
- Apply best practices in cybersecurity and software engineering
- Build a professional portfolio 

## ⚙ Engineering Principles

- Learning by doing
- Documentation first
- Continuous improvement

- Automation instead of manual tasks
- Security by design
